二分法

2025年1月25日 · 91 字 · 1 分钟

二分法有两个模板，使用模板一查找左端点，使用模板二查找右端点

模板一：

while (left < right) {
  int mid = (left + right) >> 1;
  if (check(mid) >= target) {
    right = mid;
  } else {
    left = mid + 1;
  }
}
  1. 将check(mid)看成右手，target看成左手，这是个右手不断往左手靠近的过程，直到右手在左手的左侧，因此这是求左边界，此时的右手就是左边界
  2. 循环条件是 left < right，不是 left <= right
  3. 只有当 left = mid 时，才会在计算mid时加1

模板二：

while (left < right) {
  int mid = (left + right + 1) >> 1;
  if (check(mid) <= target) {
    left = mid;
  } else {
    right = mid - 1;
  }
}
  1. 将check(mid)看成是左手，target看成是右手，这是左手不断往右手靠近的过程，直到左手在右手的右侧，因此这是求右边界，此时的左手就是右边界
  2. 循环条件 left < right，不是left <= right
  3. 只有当 left = mid时，才会在计算mid时加1
